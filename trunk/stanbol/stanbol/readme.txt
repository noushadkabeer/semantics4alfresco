To install stanbolIntegration.amp
1. shutdown alfresco server
2. copy stanbolIntegration.amp to <install_dir>/amps
3. run <install_dir>/apply_amps.bat  (pressing any key couple of times)
4. restart alfresco server

This installs:
1. Stanbol auto tagging service
2. web client auto tagging action dialog for web client
3. stanbol integration webscripts

Separate install
Also install the share amp for Alfresco Share UI for this Alfresco Apache Stanbol integration.

FlexSpaces can also be used (when an update is made to it for Apache Stanbol)
